# EvilHop

EvilHop is a .NET library for interacting with [HIP archive](https://heavyironmodding.org/wiki/EvilEngine/HIP_(File_Format)) files present in several games by video game developer Heavy Iron Studios. Namely, these games are:

* Scooby-Doo! Night of 100 Frights
* SpongeBob SquarePants: Battle for Bikini Bottom
* The SpongeBob SquarePants Movie
* The Incredibles
* The Incredibles: Rise of the Underminer
* Ratatouille ([Jan 18, 2006 prototype](https://hiddenpalace.org/Disney-Pixar_Ratatouille_(Jan_18,_2006_prototype)))
